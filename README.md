## ⚡ Quick Start

```bash
# Install dependencies
pnpm install

# Start development server (http://localhost:3000)
pnpm run dev

# Build for production
pnpm run build

# Start production server
pnpm run start
```

## 🌟 Key Features

- 🚀 **Lightning-fast SSR** - Astro's server-side rendering
- ⚛️️ **React Components** - Interactive UI elements
- 📊 **Multi-tenant Architecture** - Space and team management
- 🔐 **Secure JWT Auth** - HTTP-only cookies
- 📊 **Built-in Analytics** - Tracking and observability
- 🌙 **Modern Dark Theme** - Beautiful default UI
